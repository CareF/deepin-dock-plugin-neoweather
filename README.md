NeoWeather: A DDE Dock Plugin
==============================

### Installation Guide ###
For now there's no binary release provided. You can compile it either using Qt Creator or by the following command: 

```
mkdir build
cd build
qmake ../
make
```

And than install

```
sudo make install
```
